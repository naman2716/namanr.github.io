Cervical Fracture Detection

We utilize a dataset of cervical spine CT scans provided by the Radiological Society of North America (RSNA). This dataset comprises 3,000 studies of individual patients, used collectively for training and testing. Among these, 83 studies include segmentation data, and an additional number of studies contain bounding box coordinate data. The dataset is divided into 90% for training and 10% for validation. We employ EfficientNetV2 to learn the segmentation tasks and the YOLOv5 model to detect fractures and draw bounding boxes around the fracture areas.

Distributed File Management System using Blockchain

The Distributed File Management System uses blockchain technology as a secure and decentralized database to manage and store files, addressing issues like data tampering and unauthorized access. It employs a reverse tree structure for data integrity and comprises servers, nodes, and access points for robust storage and transfer. Built from scratch with custom blockchain architecture and a mixed cipher encryption protocol, the system ensures high data security and efficient connections over a private network. Future plans include supporting more file types, developing a user interface, and creating a mobile version, showcasing blockchain's potential in distributed file management.
